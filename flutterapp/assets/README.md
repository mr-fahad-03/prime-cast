# PrimeCast Flutter App Assets

Place your app assets here:

## images/
- logo.png - App logo
- Other images as needed

## fonts/
If you want to use custom fonts (Poppins), download and place them here:
- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf

You can download Poppins font from Google Fonts: https://fonts.google.com/specimen/Poppins

Note: If you don't add the fonts, the app will use the default system font.
